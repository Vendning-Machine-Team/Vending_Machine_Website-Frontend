## Tech Stack

**Frameworks**: vite, tailwindcss  

## Basic information
The Vending Machine Website Frontend provides a user-friendly interface for checkout, recieving a one-time code, and reporting issues.
It also supports an administrative dashboard to allow admins to update inventory/robot path and view recent changes. The frontend was built 
with vanilla HTML and incorporates tailwindcss to provide a more appealing interface. 

## Features
-API calls to recieve infromation from the database.    
-Administrative dashboard to assist admins with inventory/robot path changes.  
-Appealing user interface.   
-Stripe payment integration.
